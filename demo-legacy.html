Flexy Boxes

Perhaps the most flexible way of organizing content on your site ever

Flexbox uses container elements (flex-container) in which item elements (flex-item) can be freely positioned. Nearly any HTML element can be turned into a flexbox container or item. Flex items can be aligned horizontaly and verticaly, distribute white space around them and stretch on or both directions to use available space.

Flexbox is supported on all modern browsers with graceful fallback to a functional but less feature rich basemodel on older browsers.
